## [704. Binary Search](https://leetcode.com/problems/binary-search/description/)
**Difficulty:** Easy | **Topic:** Binary Search 

#### Complexity
* **Time:** $O(logn)$
* **Space:** $O(logn)$

#### Notes
1. Iterate while l <= r
2. When updating pointers, skip past the mid pointer

#### Code
```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        l = 0
        r = len(nums) - 1

        while l <= r:
            mid = (r - l) // 2 + l
            mid_num = nums[mid]

            if mid_num == target:
                return mid
            if mid_num < target:
                l = mid + 1
            else:
                r = mid - 1

            
        return -1
```
