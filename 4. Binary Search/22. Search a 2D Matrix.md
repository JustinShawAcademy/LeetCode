## [74. Search a 2D Matrix](https://leetcode.com/problems/search-a-2d-matrix/description/)
**Difficulty:** Medium | **Topic:** Binary Search 

#### Complexity
* **Time:** $O(logn)$
* **Space:** $O(logn)$

#### Notes
1. Reduce this problem to binary search problem by converting BS array index to matrix cell index.

#### Code
```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix: 
            return False

        num_rows = len(matrix)
        num_cols = len(matrix[0])

        # Converts flatten array index to matrix index
        def cell_index(index):
            return (index // num_cols, index % num_cols)

        # Binary search
        l = 0 
        r = num_rows * num_cols - 1

        while l <= r:
            mid = (r - l) // 2 + l
            mid_cell_index = cell_index(mid)
            mid_num = matrix[mid_cell_index[0]][mid_cell_index[1]]

            if mid_num == target:
                return True
            if mid_num < target:
                l = mid + 1
            else: 
                r = mid - 1
        
        return False
```
