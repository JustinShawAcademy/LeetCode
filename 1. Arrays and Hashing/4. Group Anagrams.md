## [4. Group Anagrams](https://leetcode.com/problems/group-anagrams/description/)
**Difficulty:** Meium | **Topic:** Dictionary 

### Approach 1: Dictionary
#### Intuition
Identify anagrams by sorting the characters in each string.

#### Complexity
* **Time:** $O(n)$
* **Space:** $O(n)$

#### Code
```python
from collections import defaultdict

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        groups = defaultdict(list)
        for s in strs:
            sorted_str = str(sorted(s))
            groups[sorted_str].append(s)

        return list(groups.values())
```
