## [167. Two Sum II - Input Array Is Sorted](https://leetcode.com/problems/two-sum-ii-input-array-is-sorted/submissions/1856378790/)
**Difficulty:** Medium | **Topic:** Two Pointers 

#### Complexity
* **Time:** $O(n)$
* **Space:** $O(1)$

#### Code
```python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        # Set up left and right pointers at the head and tail of the list
        l, r = 0, len(numbers) - 1

        while l <= r:
            two_sum = numbers[l] + numbers[r]
            if two_sum == target:
                return [l + 1, r + 1] # 1-indexed results
            
            # Manipulate l, r pointers towards the target
            if two_sum > target:
                r -= 1
            else: 
                l += 1
            
        return []
```
