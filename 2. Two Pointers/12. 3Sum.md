## 15. [3Sum](https://leetcode.com/problems/3sum/description/)
**Difficulty:** Medium | **Topic:** Two Pointers

#### Complexity
* **Time:** $O(n^2)$
* **Space:** $O(1)$

#### Intuition
1. Reduce 3Sum into 2Sum problem by looping through list.
2. Prevent recording duplicates by carefully skipping duplicates in the sorted list.
3. To find all matches, once a match is found, move each pointer inward for the next match.

#### Code
```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        res = []

        if not nums:
            return []

        # Early return: no negative numbers
        if nums[0] > 0:
            return []

        for i, num in enumerate(nums):
            # Skip duplicated nums
            if i > 0 and num == nums[i-1]:
                continue

            # Set up 2 pointers
            l, r = i + 1, len(nums) - 1

            while l < r:
                # Manipulate 2 pointers
                three_sum = num + nums[l] + nums[r]
                if three_sum == 0:
                    res.append([nums[i], nums[l], nums[r]])
                    l += 1
                    r -= 1

                    # Skips duplicates
                    while nums[l] == nums[l-1] and l < r:
                        l += 1
                elif three_sum < 0:
                    l += 1
                else: 
                    r -= 1

        return res
```

#### Alternatives
HashMap approach can also work by converting 3Sum to 2Sum problem.
