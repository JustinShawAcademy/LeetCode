## [125. Valid Palindrome](https://leetcode.com/problems/valid-palindrome/description/)
**Difficulty:** Easy | **Topic:** Two Pointers 

#### Complexity
* **Time:** $O(n)$
* **Space:** $O(1)$

#### Code
```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        l = 0
        r = len(s) - 1

        while l <= r:
            # Check indexes are pointing at alphanumeral values
            if not s[l].isalnum():
                l += 1
                continue
            if not s[r].isalnum():
                r -= 1
                continue
            
            # Compare alphanumeric values at l and r. make sure to compare lower cased letter.
            if s[l].lower() != s[r].lower():
                return False
            
            # if values are the same, proceed to next value pairs
            l += 1
            r -= 1
            
        return True
```

#### Alternatives
1. Reverse string: this is the most intuitive solution - leave up the string and leave only the alphanumerals, then company the reversed string with the original. This takes O(n) space.
