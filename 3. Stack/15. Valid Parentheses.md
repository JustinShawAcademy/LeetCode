## [20. Valid Parentheses](https://leetcode.com/problems/valid-parentheses/description/)
**Difficulty:** Easy | **Topic:** Stack 

#### Complexity
* **Time:** $O(n)$
* **Space:** $O(n)$

#### Code
```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []

        for char in s:
            if not stack or char in ["(", "[", "{"]:
                stack.append(char)
            else: # handle closing brackets
                prev_char = stack.pop()
                if prev_char == "(" and char == ")" \
                    or prev_char == "[" and char == "]" \
                    or prev_char == "{" and char == "}":
                    continue
                else: 
                    return False
        
        return not stack
```
