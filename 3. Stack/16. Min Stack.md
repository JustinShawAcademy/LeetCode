## [155. Min Stack](https://leetcode.com/problems/min-stack/description/)
**Difficulty:** Meidum | **Topic:** Stack 

#### Complexity
* **Time:** $O(n)$
* **Space:** $O(n)$

#### Intuition
Use 2 stacks:
* One stack tracks the original order of elements
* The other stack tracks the min element for every push

#### Code
```python
class MinStack:

    def __init__(self):
        self.stack = []
        self.min_stack = []

    def push(self, val: int) -> None:
        self.stack.append(val)
        
        if not self.min_stack or val <= self.min_stack[-1]:
            self.min_stack.append(val)

    def pop(self) -> None:
        val = self.stack.pop()
        if self.min_stack and self.min_stack[-1] == val:
            self.min_stack.pop()        

    def top(self) -> int:
        return self.stack[-1]

    def getMin(self) -> int:
        return self.min_stack[-1]
```
