## [739. Daily Temperatures](https://leetcode.com/problems/daily-temperatures/description/)
**Difficulty:** Medium | **Topic:** Stack 

#### Complexity
* **Time:** $O(n)$
* **Space:** $O(n)$

#### Intuition
1. Use a stack to track days before a new high temperature day is found

#### Code
```python
class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        # Create a class for better readability
        class DayTemperature:
            def __init__(self, day, temperature):
                self.day = day
                self.temperature = temperature

        stack = []
        res = [0] * len(temperatures) # init array for random access

        for i, temperature in enumerate(temperatures):
            while stack and stack[-1].temperature < temperature:
                # find the next high tempature day
                prev_day = stack.pop()
                res[prev_day.day] = i - prev_day.day

            # append current day
            stack.append(DayTemperature(i, temperature))

        return res

```
